## 🚀 사용 방법

### 환경변수 설정 (.env.local)
```env
NEXT_PUBLIC_USE_AI_VERIFICATION=true
AI_BACKEND=ollama
AI_BASE_URL=http://localhost:11434
AI_MODEL_ANALYSIS=gemma3:4b
```

### Ollama 확인
```powershell
# Ollama 연결 확인
Invoke-RestMethod -Uri "http://localhost:11434/api/tags"

# 모델 테스트
ollama run gemma3:4b "안녕하세요"
```

### 전략 실행
1. 자동 매매 섹션에서 전략 선택
2. 변동성 돌파 또는 모멘텀 선택
3. 파라미터 설정:
   - **변동성 돌파**: 승수 (0.1-2.0, 기본값: 0.5)
   - **모멘텀**: 기간 (5-30일), 임계값 (1-20%, 기본값: 5%)
4. 전략 추가 클릭

## 📈 AI 검증 작동 방식

모든 전략은 AI 검증을 통해 다음을 수행합니다:

1. **신호 평가**: 기술적 지표 기반 매매 신호를 AI가 평가
2. **차단 기능**: AI가 부정적 판단 시 거래 건너뛰기
3. **신뢰도 기반**: confidence > 0.6-0.7 이상일 때만 실행
4. **Fallback**: AI 실패 시 기본 전략 로직 실행

## 🔍 전략별 AI 검증 상세

### 변동성 돌파
- AI가 가짜 돌파(False Breakout) 판단
- 유효한 돌파로 판단되면 매수 실행
- 신뢰도 임계값: 0.6

### 모멘텀
- 가격 모멘텀과 거래량 모멘텀 종합 평가
- 모멘텀 지속 가능성 평가
- 신뢰도 임계값: 0.65

### MA/RSI/볼린저밴드
- 기술적 지표 신호를 AI가 종합 평가
- 추세와 지표 불일치 시 차단
- 신뢰도 임계값: 0.7
