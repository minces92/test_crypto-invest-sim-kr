# Component: `Header`

## 1. 개요

애플리케이션의 최상단에 위치하여 제목과 부제를 표시하는 헤더 컴포넌트입니다.

## 2. 기능

-   애플리케이션의 이름인 "Crypto Invest Sim"을 큰 제목으로 보여줍니다.
-   "가상 암호화폐 투자 시뮬레이션"이라는 부제를 작은 글씨로 함께 표시합니다.

## 3. 주요 상태(State) 및 Props

이 컴포넌트는 외부로부터 전달받는 `props`나 자체적으로 관리하는 `state`가 없는 순수 정적(static) 컴포넌트입니다.

## 4. 구조

-   `<header>` HTML 태그를 사용하여 시맨틱한 구조를 가집니다.
-   Primer CSS 클래스를 사용하여 스타일링되었습니다.
