# Crypto Invest Sim KR

## 1. 프로젝트 개요

- **이름:** `crypto-invest-sim-kr`
- **목표:** 사용자가 가상의 자본으로 실제 암호화폐 시세를 기반으로 매수/매도 거래를 시뮬레이션할 수 있는 웹 애플리케이션을 제작합니다. 모든 UI와 금액 표시는 한국 사용자에게 맞춰 한글과 원화(KRW)를 기본으로 합니다.

## 2. 주요 기능

1.  **실시간 시세 조회:** 한국 암호화폐 거래소(예: 업비트, 빗썸)의 API를 이용하여 주요 암호화폐의 현재 시세를 원화(KRW)로 표시합니다.
2.  **가상 거래:** 사용자에게 초기 가상 자본(예: 1,000,000원)을 부여하고, 이를 이용해 암호화폐를 매수하거나 매도할 수 있습니다.
3.  **포트폴리오 관리:** 사용자가 보유한 암호화폐 목록, 수량, 평균 매수 단가, 현재 평가 금액 및 수익률을 보여주는 대시보드를 제공합니다.
4.  **반응형 UI:** 모바일 기기에서도 UI가 깨지지 않고 편안하게 사용할 수 있도록 반응형으로 디자인되었습니다.
5.  **빠른 데이터 갱신:** 실시간 시세 데이터를 5초 주기로 갱신하여 사용자가 더 빠르게 시장 변화에 대응할 수 있습니다.

## 3. 사용 기술 (Technology Stack)

- **프레임워크:** Next.js (React 기반)
- **언어:** TypeScript
- **스타일링:** Bootstrap
- **데이터 소스:** 국내 거래소의 Public API (예: Upbit API)

## 4. 문서 (Documentation)

프로젝트의 상세한 기술 문서, API 명세 등은 [docs](./docs/README.md) 디렉토리에서 확인할 수 있습니다.

## 5. 개발 단계

1.  **프로젝트 초기 설정:** `create-next-app`을 사용하여 Next.js와 TypeScript 기반의 프로젝트 구조를 생성합니다.
2.  **기본 UI 컴포넌트 설계:**
    -   `Header`: 앱 제목과 전체 자산 정보를 표시합니다.
    -   `CryptoTable`: 실시간 시세 정보를 보여주는 테이블입니다.
    -   `Portfolio`: 보유 자산 현황을 보여주는 컴포넌트입니다.
    -   `TradeModal`: 매수/매도 주문을 입력하는 모달 창입니다.
3.  **API 연동:** Next.js의 API Routes 기능을 사용하여 외부 암호화폐 거래소 API와 통신하는 백엔드 로직을 구현합니다.
4.  **상태 관리:** React Context API 또는 간단한 `useState` 훅을 사용하여 사용자의 가상 자본, 보유 암호화폐 등 클라이언트 측 상태를 관리합니다.
5.  **기능 구현 및 통합:** 위 요소들을 통합하여 실제 거래 로직(매수/매도), 포트폴리오 업데이트 등을 구현하고 전체 애플리케이션을 완성합니다.
