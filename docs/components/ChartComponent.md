# Component: `ChartComponent`

## 1. 개요

특정 암호화폐 마켓의 최근 30일간의 가격 변동을 보여주는 라인 차트 컴포넌트입니다.

## 2. 기능

-   `market` prop으로 전달받은 마켓 코드에 해당하는 일봉(day candle) 데이터를 조회합니다.
-   `/api/candles` API 엔드포인트를 호출하여 차트 데이터를 가져옵니다.
-   `recharts` 라이브러리를 사용하여 시계열 라인 차트를 렌더링합니다.
-   데이터를 불러오는 동안 "차트 로딩 중..." 메시지를 표시합니다.
-   차트의 X축은 날짜(월/일), Y축은 가격(원화)을 나타냅니다.
-   툴팁(Tooltip)에 마우스를 올리면 해당 날짜의 정확한 가격을 원화 형식으로 보여줍니다.

## 3. 주요 상태(State) 및 Props

### Props

| Prop     | 타입   | 필수 | 설명                                  |
| -------- | ------ | ---- | ------------------------------------- |
| `market` | string | 예   | 차트를 표시할 암호화폐 마켓 코드 (예: `KRW-BTC`) |

### 내부 상태

-   `data`: API로부터 받아온 캔들 데이터 배열 (`Candle[]`)
-   `loading`: 데이터 로딩 상태를 관리하는 boolean 값

## 4. 데이터 흐름

1.  컴포넌트가 렌더링되면 `useEffect` 훅이 실행됩니다.
2.  `market` prop이 변경될 때마다 `/api/candles?market={market}&count=30` API를 호출합니다.
3.  API 응답으로 받은 데이터는 시간 순서대로 정렬하기 위해 배열 순서를 뒤집습니다 (`rawData.reverse()`).
4.  가져온 데이터를 `data` 상태에 저장하고 `loading` 상태를 `false`로 변경합니다.
5.  `data` 상태가 업데이트되면 `recharts`의 `LineChart`가 차트를 다시 렌더링합니다.
