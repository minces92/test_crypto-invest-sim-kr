# AI 추천 전략 마켓 선택 버그 수정

## 문제점

1.  **AI 추천 기능 오류**: '자동 매매' 컴포넌트('나만의 전략' 탭)에서 'AI 추천 받기' 버튼을 클릭해도 항상 "마켓을 먼저 선택해주세요"라는 오류 메시지가 발생하며 기능이 동작하지 않았습니다.
2.  **잘못된 상태 변수 사용**: UI의 '코인 선택' 드롭다운은 `market` 상태 변수를 업데이트하지만, AI 추천을 요청하는 핸들러(`handleGetRecommendation`)는 `selectedMarket`이라는 별개의 초기화되지 않은 상태 변수를 참조하고 있었습니다. 이로 인해 사용자가 마켓을 선택하더라도 로직에서는 선택되지 않은 것으로 판단했습니다.

## 수정 내용

### `src/components/AutoTrader.tsx` 개선

-   **상태 변수 통합**: 불필요하고 사용되지 않는 `selectedMarket` 상태 변수를 코드에서 완전히 제거했습니다.
-   **핸들러 로직 수정**:
    -   `handleGetRecommendation` 함수가 올바른 `market` 상태 변수를 참조하여 AI 추천 API를 호출하도록 수정했습니다.
    -   `handleStartAIStrategy` 함수 또한 `market` 상태 변수를 사용하도록 수정하여, 추천받은 전략을 적용할 때 올바른 마켓 정보가 사용되도록 변경했습니다.
-   **사용자 경험(UX) 개선**: 이제 사용자는 '나만의 전략' 탭에 있는 '코인 선택' 드롭다운에서 마켓을 선택하고, 바로 AI 추천 기능을 사용할 수 있어 직관적인 흐름이 완성되었습니다.

## 변경된 파일

-   `src/components/AutoTrader.tsx`: AI 추천 로직의 상태 관리 버그 수정

## 테스트 방법

1.  애플리케이션의 '자동 매매' 섹션으로 이동합니다.
2.  '나만의 전략' 탭을 선택합니다.
3.  '기본 정보' 섹션에 있는 '코인 선택' 드롭다운에서 원하는 마켓(예: ETH)을 선택합니다.
4.  'AI 전략 추천' 섹션의 'AI 추천 받기' 버튼을 클릭합니다.
5.  선택한 마켓(ETH)에 대한 추천 전략이 오류 없이 성공적으로 로드되는지 확인합니다.
6.  추천받은 전략의 '이 전략 적용하기' 버튼을 눌러 정상적으로 활성 전략 목록에 추가되는지 확인합니다.

## 주의사항

-   AI 전략 추천을 위한 마켓 선택은 이제 '나만의 전략' 탭 내의 '코인 선택' 드롭다운을 통해 이루어집니다.
